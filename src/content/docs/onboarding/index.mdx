# Onboarding

This is a suggested path for new members to develop the skills to contribute meaningfully to robotics projects.

## Before you start

- At least 50 GB free disk space.
- Stable internet connection.
- A few hours of time (and power if on battery) you can keep your computer running.
- A GitHub account.
- Potentially a USB drive if you plan to dual-boot Ubuntu.
- A decent computer made in the last decade.

If Linux or Git are new to you, keep a [Linux command reference](https://ubuntu.com/tutorials/command-line-for-beginners) and the [Git docs](https://git-scm.com/docs) open while you work.

---

## Guided Steps

Follow these steps in order to set up your environment and make your first contribution.

1.  **[Step 1: Choose your platform](./platform-choice)** — Get into an Ubuntu 24.04 environment.
2.  **[Step 2: Install ROS 2 Jazzy](./ros-install)** — Automated or manual installation of core tools.
3.  **[Step 3: Verify your install](./verify)** — Ensure everything is working correctly.
4.  **[Step 4: Run a ROS 2 demo](./first-demo)** — Your first "Hello World" in ROS 2.
5.  **[Step 5: PX4 flight test](./px4-test)** — Full SITL simulation with Gazebo and ROS 2.
6.  **[Step 6: Make your first contribution](./contribute)** — Submitting your first Pull Request.
