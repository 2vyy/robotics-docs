We use an automated script to install ROS 2, Gazebo, and PX4 development tools. This script sets up a clean, consistent environment.

### Preferred Path: Automated Script

The installation script handles all system dependencies automatically. If the installation is interrupted (e.g., connection issues), run it again; it safely resumes from where it left off.

**Install steps:**

```bash
# TODO: replace with the actual raw URL once the repo is public
# e.g. https://raw.githubusercontent.com/your-org/your-repo/main/robotics-setup.sh
curl -sSL <SCRIPT_URL> -o robotics-setup.sh
chmod +x robotics-setup.sh

# Run full installation
./robotics-setup.sh --full
```

The script performs the following tasks:
- Configures Ubuntu repositories for ROS 2.
- Installs ROS 2 Jazzy, Gazebo Harmonic, and PX4 SITL tools.
- Creates a Python virtual environment for necessary libraries.
- Configures your `~/.bashrc` to auto-source ROS 2 and your workspace.

:::caution WSL2 — Gazebo display requirements
Gazebo requires a working OpenGL context. On WSL2, install the latest GPU driver on the **Windows side** (not inside WSL) from your GPU vendor (NVIDIA/AMD/Intel). After installing, restart WSL with `wsl --shutdown` in PowerShell before launching Gazebo.

If Gazebo still won't open, you can run headless simulations with `gz sim -s` (server-only) and connect a remote client later.
:::

Once finished, proceed to [Step 3](./verify).

### Manual Reference

If you need to debug your setup, the script performs the following operations in order:

1. **System Setup**: Updates package lists and ensures a UTF-8 locale.
2. **ROS 2 Jazzy**: Adds the official ROS 2 apt repository and installs `ros-jazzy-desktop`.
3. **Gazebo**: Installs `ros-jazzy-ros-gz` for integration between ROS 2 and Gazebo Harmonic.
4. **PX4 SITL**: Clones `PX4-Autopilot`, installs build dependencies, and compiles the firmware.
5. **Micro XRCE-DDS**: Compiles the bridge agent from source to connect PX4 and ROS 2.
6. **Python Environment**: Sets up a virtual environment in `~/ros2_venv` with required packages like `numpy`, `mavsdk`, and `opencv-python-headless`.

To create your own packages after installation:

```bash
cd ~/ros2_ws/src
ros2 pkg create <your_package_name> --build-type ament_python --dependencies rclpy ...
```
